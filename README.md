# Week 1 — User Research & UX Analysis

## Project

Design research for a **Fitness App** targeted at busy working professionals who struggle to maintain a healthy lifestyle because of demanding work schedules.

### Objective

Understand users' daily routines, challenges, motivations, and expectations before designing a solution.

### Activities Completed

- User Research
- User Interviews
- Empathy Mapping
- User Persona

### Key Insights

Through user interviews, I identified several common problems among working professionals:

- Long working hours reduce time available for exercise.
- Existing fitness apps often require lengthy workout sessions.
- Users want quick and practical fitness routines.
- Maintaining consistency is more difficult than starting.

These findings helped me understand the users' pain points and design opportunities before moving into the solution phase.

# Week 3 — High Fidelity Mobile UI Design

## Project

Design a mobile application for an **Organic Vegetable Delivery Service** that connects local farms directly with customers.

Unlike traditional grocery delivery platforms, this concept focuses specifically on delivering fresh, organically grown vegetables in a clean and trustworthy shopping experience.

### Objective

Transform research and design principles into a polished, high-fidelity mobile interface.

### Deliverables

- Moodboard
- Design System
- Splash Screen
- Home Screen
- Product Detail Screens
- Interactive Prototype

### Design Direction

The application was designed around four core values:

- Freshness
- Trust
- Simplicity
- Healthy Living

To reflect these values, I used:

- Warm earthy color palette
- Natural farm imagery
- Clean typography
- Rounded UI components
- Minimal interface
- High-quality product photography

The final design provides users with a calm and intuitive shopping experience while reinforcing the authenticity of farm-fresh organic products. The home screen includes product browsing, categories, seasonal recommendations, and a "Buy Again" feature for frequently purchased vegetables.
